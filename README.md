# オーディオブック購入予測モデル

## 1. 概要

本プロジェクトでは、顧客の過去の行動データをもとに、オーディオブックを購入するかどうかを予測する二値分類モデルを構築しました。

顧客の利用履歴や行動指標から購買傾向を学習し、マーケティング施策やレコメンドシステムへの活用を想定しています。

---

## 2. 使用技術

- Python
- NumPy
- TensorFlow / Keras
- scikit-learn（前処理）
- Jupyter Notebook

---

## 3. データについて

- CSV形式の顧客行動データを使用
- 説明変数：顧客の利用履歴・行動指標
- 目的変数：
  - 0：購入しない
  - 1：購入する

データにはクラス不均衡（非購入者が多い）が存在していました。

---

## 4. 前処理

### ① クラス不均衡の調整
非購入データを一部削除し、購入者と非購入者の数を均等化。

### ② 標準化
各特徴量を平均0・分散1にスケーリング。

### ③ データ分割
- 学習データ：80%
- 検証データ：10%
- テストデータ：10%

### ④ シャッフル処理
学習の偏りを防ぐためランダムに並び替え。

---

## 5. モデル構成

多層パーセプトロン（MLP）による二値分類モデルを構築。

構造：

- 入力層
- 隠れ層：50ユニット（ReLU）
- 隠れ層：50ユニット（ReLU）
- 出力層：2ユニット（Softmax）

設定：

- 損失関数：sparse_categorical_crossentropy
- 最適化手法：Adam
- EarlyStoppingによる過学習防止

---

## 6. 学習結果

テストデータに対して分類精度を評価し、顧客の購買傾向を一定の精度で予測可能であることを確認しました。

---

## 7. 工夫した点

- クラス不均衡問題に対応するためのデータバランス調整
- 標準化による学習の安定化
- EarlyStoppingを用いた過学習対策
- 学習・検証・テストの適切なデータ分割による汎化性能の確保

データ前処理、クラス不均衡への対応、過学習対策など、実務を意識した機械学習パイプラインを構築しています。
